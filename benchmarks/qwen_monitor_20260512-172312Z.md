$$

```

为什么：

- **Self-Attention 擅长捕获数据/特征的内部相关性**，Multi-head Self-attention 又能进一步增强这种表示能力。[Source: ch4(2).pdf, Page 41/97]

简言之：**RNN 是按时间递推地传递信息，Transformer 则用 Self-Attention 让序列位置之间直接交互，因此更利于建模全局上下文。**
```

**Answer:**

```
The **Markov assumption** says that in a sequence, the probability of the next state depends only on the current/previous state, not on the full history of earlier states.

In the HMM/Viterbi recursion, the transition into state `$i$` uses only the previous state `$j$` through the transition probability `$a_{ji}$`, which reflects this “one-step dependence” idea. [Source: ch3.pdf, Page 32/51] HMMs are described as probability functions of a Markov process, which is the modeling basis for this assumption. [Source: ch3.pdf, Page 2/51]
```
